This server provides several endpoints for different types of connections:

GET /

Root endpoint - serves this information page for browsers, or JSON data for API clients.

GET /sse

Server-Sent Events endpoint for real-time communication with the MCP server.

GET /mcp

Main MCP endpoint for establishing connections with session management.

POST /mcp

For sending MCP requests to the server.

POST /message

For sending messages to existing SSE sessions.

ðŸ”§ How to Connect

To connect to this MCP server, use the following configuration in your MCP client:

{ "mcpServers": { "playwright": { "url": "YOUR_SERVER_URL/mcp", "headers": { "Authorization": "Bearer YOUR_APIFY_TOKEN" } } } }

Note: Replace YOUR_SERVER_URL with the actual server URL and provide your Apify token for authentication.
